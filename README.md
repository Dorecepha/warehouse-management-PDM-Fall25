### SPECIFICATION
 - Require Open JDK 25 and Java 21
 - To run the project, you will also need to install [MySQL](https://dev.mysql.com/downloads/installer/). [How to install](https://www.youtube.com/watch?v=hiS_mWZmmI0&t=187s)
 - Also, for backend I recommend [IntelliJ IDEA](https://www.jetbrains.com/idea/), for frontend I would recommend using [WebStorm](https://www.jetbrains.com/webstorm/) or [VS Code](https://code.visualstudio.com/)
